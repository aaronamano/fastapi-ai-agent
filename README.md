## Overview
[https://youtu.be/eSbeub2ZeNk?si=weXtyqePy0jQaYvD](Build and Deploy a Python AI Agent) by Tech with Tim

## Setup
```bash
conda create -n langchain-env python=3.11 -y
conda activate langchain-env
pip install requirements.txt
```

## Running the app
```bash
conda activate langchain-env
python main.py
```
